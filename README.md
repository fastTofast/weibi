# weibi 微笔
随时随地记录，高效简洁
##使用技术栈
  ###前端：uni-app(基于vue.js,vuex,vue-router,mpvue,5+app)
  ###后端：Koa2,mongodb,mongoos
  ###服务器端：Nginx,pm2
 
 ##主要功能
    ###用户注册，登录功能
    ###记录发布，可选择是否共享
    ###展示共享的记录
    ###展示关注的记录（未完工）
    ###展示自己的记录
    ###展示收藏的记录（未完工）
    ###权限控制
    ###对自己的记录进行修改
    ###对自己的记录进行删除
    ###个人页信息展示，可编辑签名
